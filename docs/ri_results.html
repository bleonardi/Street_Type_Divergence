Overview

The hypothesis: street suffix types (Court, Boulevard, Road, etc.) originally carried typological meaning — they described what a street actually was. Over the post-war suburban era, developers repurposed evocative suffixes as marketing vocabulary, decoupling the name from the built form.

This analysis tests that empirically for Rhode Island using OpenStreetMap data, cross-referencing suffix against four built-form attributes:

  • Highway classification (OSM highway=* tag, ranked from service road → motorway)
  • Lane count
  • Posted speed limit
  • Sidewalk presence
  • Intersection density (intersections per km — high = connected grid, low = cul-de-sac/dead-end)

A divergence score (0–1) measures how far each suffix’s observed median profile sits from its pre-modern archetype.

Divergence scores

Each suffix is scored against a typological archetype — the built-form profile implied by its original meaning (e.g., a Court should be short, low-speed, low-connectivity; a Boulevard should be wide, multi-lane, tree-lined and pedestrian-accessible).

    "Parkway":   dict(hw=4, lanes=4,   speed=40, sidewalk=True,  intx=3),

profile = (
    df_named.groupby("suffix").agg(
        n            = ("osm_id", "count"),
        hw_rank      = ("hw_rank", "median"),
        lanes        = ("lanes", "median"),
        maxspeed     = ("maxspeed", "median"),
        sidewalk_pct = ("has_sidewalk", "mean"),
        intx_density = ("intersection_density", "median"),
        way_length_m = ("way_length_m", "median"),
    ).reset_index()
)

def divergence_score(row):
    arch = ARCHETYPES.get(row["suffix"])
    if arch is None:
        return np.nan
    deltas = []
    if not np.isnan(row["hw_rank"]):
        deltas.append(abs(row["hw_rank"] - arch["hw"]) / 6)
    if not np.isnan(row["lanes"]):
        deltas.append(abs(row["lanes"] - arch["lanes"]) / 5)
    if not np.isnan(row["maxspeed"]):
        deltas.append(abs(row["maxspeed"] - arch["speed"]) / 50)
    if not np.isnan(row["sidewalk_pct"]):
        deltas.append(abs(row["sidewalk_pct"] - (1.0 if arch["sidewalk"] else 0.0)))
    if not np.isnan(row["intx_density"]):
        deltas.append(abs(row["intx_density"] - arch["intx"]) / 15)
    return np.mean(deltas) if deltas else np.nan

profile["divergence"] = profile.apply(divergence_score, axis=1)
profile = profile.sort_values("divergence", ascending=False)

Key findings — Rhode Island

Highest divergence (name has drifted far from form):

  • Parkway — median 2 lanes, ~55 mph. In RI this is essentially highway classification with a bucolic name. The “park” in parkway is long gone.
  • Boulevard — median 2 lanes, 30 mph. Supposed to imply a wide, tree-lined multi-lane road with pedestrian amenity. In RI it’s functionally a Street with better branding.
  • Court & Place — supposed to be quiet, low-connectivity residential dead-ends. Intersection density suggests they’re not much more isolated than a generic Drive.
  • Lane — the quintessential “small residential side street” is pulling 25 mph / 2 lanes, indistinguishable from Street on most attributes.

Lowest divergence (name still predicts form):

  • Road — the most honest suffix. Generic, rural, utilitarian — and OSM agrees.
  • Turnpike / Pike — preserved their functional identity as through-routes (higher speed, limited intersections).
  • Trail — retains some distinctiveness (lower speeds, narrower), possibly because trail names are more often applied to actual paths/greenways in OSM.
  • Run — small sample but consistent with a rural/semi-rural character.

The convergence cluster: Street, Drive, Lane, Court, Place, Circle, and Way are nearly indistinguishable from each other on all measured attributes. They have collapsed into a single suburban residential archetype — 2 lanes, 25 mph, low sidewalk coverage, moderate connectivity. The suffix carries no predictive information about built form.

Caveats & next steps

  • OSM tagging completeness: Speed and lane tags are sparse in RI — many ways have no maxspeed or lanes entry, so profiles lean heavily on highway rank and connectivity. National coverage will improve this.
  • Intersection density as a metric: The current measure counts any shared OSM node as an intersection, including driveways and minor service connections. A cleaned version should filter to highway=* intersections only.
  • Geographic scope: RI is dense, urban, and historically street-grid-heavy (Providence). Results may look different in sprawl-dominant metros — expanding to the full US is the critical next step.
  • Temporal angle: Adding parcel/subdivision vintage data could test whether divergence is a post-1950 phenomenon.
